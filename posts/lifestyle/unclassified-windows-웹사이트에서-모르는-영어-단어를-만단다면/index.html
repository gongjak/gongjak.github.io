[Windows] 웹 사이트에서 모르는 영어 단어를 만난다면...

August 28, 2006 · 1 분

외국 사이트를 돌아다니다 보면모르는 단어들이 군데 군데 있을 것입니다.이를 매번 사전을 찾아보기도 힘들고…편하다는 컴퓨터의 영어사전이나 인터넷 사전도 단어를 입력하거나검색해오는 시간이 걸립니다.

http://toolbar.google.com/T4/intl/ko/index.html 에서 구글 툴바를 설치해보세요~~~툴바의 번역 버튼이 켜져있다면,단순히 모르는 영어 단어위에 커서를 올리기만 하면 단어의 뜻이 나옵니다.아주 편하죠??? ㅎㅎ

새롭게 강화된 구글 툴바는 구글에 로그인을 하시면구글 툴바가 설치된 컴에서는 자신만의 즐겨찾기를 언제든지 사용하실 수 있습니다.컴의 OS를 다시 설치한다해도구글 툴바만 다시 설치하면 즐겨찾기도 그대로 되돌아 옵니다.

팝업차단 기능도 추가되어 있어서윈도우 서비스팩2의 팝업 차단기능으로도 막지 못하는보기 싫은 팝업들도 완벽하게 막아줄 수 있습니다.팝업을 보고 싶다면 팝업이 나오는 창이 열릴 때 Ctrl 키를 누르고 있어주면 열립니다.특히, 결제창의 경우 반드시 눌러줘야만 하죠…귀찮을 수도 있는데,평상시 마구 열리는 팝업창을 막을 수 있으니 마음이 편합니다.

이 외에도 여러가지 기능이 있으니 많이 사용해보세요~~~
